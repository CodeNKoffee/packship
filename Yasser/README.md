## Supported React.js

This package supports React.js `18.2.0` and above.

## Installation

To install Yasser, simply run:

```bash
npm install Yasser
```

Or with Yarn:

```bash
yarn add Yasser
```

## Usage

Basic example usage:

```jsx
import Yasser from 'Yasser';

const example = () => {
  return (
    <div>
      <Yasser />
    </div>
  );
};

export default example;
```
